# Диагностика проблемы 404

## Результаты анализа

Пост EDC доступен по URL с кириллицей:
`https://nodkeys.com/posts/чуть-про-вещи-мой-edc-и-отношение-к-функциональности/`

Проблема была в том, что URL `edc-everyday-carry` не существует — Hugo генерирует URL на основе заголовка поста на русском языке.

## Текущая структура URL
- Главная: https://nodkeys.com/
- Посты: https://nodkeys.com/posts/
- EDC пост: https://nodkeys.com/posts/чуть-про-вещи-мой-edc-и-отношение-к-функциональности/

## Рекомендация
